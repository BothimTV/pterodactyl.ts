# Security Policy

## Supported Versions

| Version  | Supported          |
| -------- | ------------------ |
| >= 1.0.0 | :white_check_mark: |
| < 1.0.0  | :x:                |

## Reporting a Vulnerability

Please report vulnerabilities via [github security advisories](https://github.com/BothimTV/pterodactyl.ts/security) or via mail: security@bothimtv.com.
